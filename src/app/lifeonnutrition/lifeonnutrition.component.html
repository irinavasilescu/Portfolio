LIFE.ON NUTRITION


Stores relevant client data such as age, height, weight, measurements, allergies, contact details.
Custom menus are created for each client, depending on their preferences, allergies and desired result.
Shopping lists are generated automatically for each client, based on the corresponding menu, for easy weekly shopping.
Tracks clients' evolution during the diet to make sure each menu given helps achieve the desired result.
Store recipes and predefined menus for fast and easy import while creating weekly menus.
Nutritional values are automatically calculated while creating menus, making it easy to associate the perfect menu to each client.
